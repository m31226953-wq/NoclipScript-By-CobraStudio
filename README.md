# Noclip Script Classic+

Простой ноклип для Roblox с GUI-индикатором.

## Управление
- Клавиша X — включить/выключить

## Особенности
- Работает только на ПК
- Отключает коллизию персонажа со всеми объектами
- GUI показывает статус ON/OFF
- Автоматически восстанавливается после респавна
